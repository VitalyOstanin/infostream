## Пример 1: Пошаговый разбор кода из документации

Промпт для пошагового разбора конкретного приёма из документации с фильтрацией примеров по технологии.

```text
Прочитай
https://github.com/VitalyOstanin/infostream/blob/master/recipes/streaming-etl-mongodb.md

давай разберём по пунктам приём 1 (async generator). По шагам, по одному логическому пункту за раз.

- пропусти все примеры для mongodb, оставь только про mongoose.
```

## Пример 2: Исследование законодательства с перекрёстной проверкой

Промпт для поиска и анализа правовой информации с требованием множественных источников и практических действий.

```text
Найди в законодательстве РФ про уменьшение стоимости патента для ИП за счёт фиксированных взносов.

На примере СПБ в 2026 году, ОКВЭД 62.01, ИП без работников.

- Проверь свой ответ, должно быть не менее трёх авторитетных источников.
- Укажи в ответе ссылки на источники.
- Укажи действия для уменьшения стоимости при обслуживании ИП в банке Точка, в ТБанке.
- Если не можешь прочитать какие-то веб-страницы через свои инструменты, открывай такие страницы в браузере через playwright.
```

## Пример 3: Настройка MCP-сервера по README проекта

Промпт для автоматической настройки инструмента по документации с интерактивным уточнением параметров и безопасной обработкой секретов.

```text
Прочитай проект https://github.com/sooperset/mcp-atlassian

Добавь в свои настройки ~/.claude.json этот mcp сервер под именем jira-mcp.
Нужны jira и confluence.

- Запрашивай у меня интерактивно ПО ОДНОЙ настройки для этого сервера, кроме секретов, их я укажу сам после первичной настройки.
- Пока что вместо секретов впиши placeholders.
- Перед записью изменений в файл покажи их мне и согласуй.
- В конце покажи мне файл, куда надо вручную вписать секреты.
```
